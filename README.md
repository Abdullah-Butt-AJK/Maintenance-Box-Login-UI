# 🔧 Maintenance Box — Login UI
 
A clean and modern **Login / Sign-Up UI** built with **Flutter**, designed for the Maintenance Box mobile application. This project focuses on UI design and user authentication flow, targeting both Android and iOS platforms.

## ✨ Features
 
- **Login Screen** — Email & password fields with a visibility toggle for the password
- **Sign Up Navigation** — Redirect prompt for new users
- **Custom Branding** — Maintenance Box logo with orange accent color theme
- **Clean UI Design** — Minimal, intuitive layout with rounded input fields
- **Cross-Platform** — Runs on Android & iOS via Flutter

## 🚀 Getting Started
 
### Prerequisites
 
Make sure you have the following installed:
 
- [Flutter SDK](https://docs.flutter.dev/get-started/install) (stable channel)
- [Android Studio](https://developer.android.com/studio) or [VS Code](https://code.visualstudio.com/)
- An Android emulator or physical device
### Installation
 
```bash
# 1. Clone the repository
git clone https://github.com/Abdullah-Butt-AJK/Maintenance-Box-Login-UI.git
 
# 2. Navigate into the project directory
cd Maintenance-Box-Login-UI
 
# 3. Install dependencies
flutter pub get
 
# 4. Run the app
flutter run
```
